# **App Name**: AquaVita

## Core Features:

- Animated Loading Sphere: Animated loading sphere that fills with 'water' and displays a loading counter.
- Animated Hero Section: Hero section with background video and animated SVG waves. Waves transition on scroll.
- Scrolling Phrase Display: Scrolling phrases section with icons that appear/disappear responsively on scroll.
- Interactive Video Tab: Interactive 'Our Purpose' tab section for video.
- Multi-language Support: Multi-language support for all pages.

## Style Guidelines:

- Primary color: White for a clean, modern feel.
- Secondary color: A specific shade of green/blue (like teal) to represent water and nature.
- Tertiary color: Black for text and important UI elements, providing contrast.
- Accent: Light Blue (#ADD8E6) to subtly highlight interactive elements and reinforce the water theme.
- Use minimalist, line-based icons to maintain a clean aesthetic.
- Ensure a responsive layout that adapts to different screen sizes.
- Implement subtle and engaging animations using GSAP throughout the website to enhance user experience. For example, smooth transitions, parallax scrolling effects, and animated SVG elements.
